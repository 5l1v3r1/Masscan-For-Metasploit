Masscan-for-Metasploit

Masscan bash Implementation for direct import to metasploit.

This project uses masscan (https://github.com/robertdavidgraham/masscan) to scan an IP block and produce result in XML format which can then be directly imported to Metasploit Framework (https://github.com/rapid7/metasploit-framework) using db_import.

By default the XML generated by masscan is not compatible with Metasploit

Installation

git clone https://github.com/tiwaryshailesh/Masscan-For-Metasploit.git
 cd Masscan-For-Metasploit
 bash install.sh

Usage

massnmap output-File-Name ipblock commaSeperatedPortRange

example

msf>   massnmap HRDept 192.168.27.1/24 80,443,23
 msf>   db_import HRDept.xml
